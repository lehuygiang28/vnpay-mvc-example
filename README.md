# vnpay-mvc-example

## Cài đặt

1. Cài đặt phụ thuộc

```bash
$ npm install
```

2. Cài đặt các biến môi trường

-   Copy file `.env.example` và đổi tên thành `.env`
-   Điền thông tin cấu hình vào file `.env`

3. Chạy ứng dụng

-   Chế độ phát triển

```bash
$ npm run dev
```

-   Chế độ production

```bash
$ npm run build
$ npm run start
```
